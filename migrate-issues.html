VaporForge Issue Migration

Migrate your issue tracker data between domains.

Step 1: Export from old domain

Open this page on vaporforge.jbcloud.app, then click Export.

Or: open vaporforge.jbcloud.app in another tab, run the console command below, and paste the result.




Step 2: Import to new domain

Open this page on vaporforge.dev, paste JSON above, then click Import.

Alternative: Direct Console Commands

On vaporforge.jbcloud.app console:

copy(localStorage.getItem('vf-issue-tracker'))

On vaporforge.dev console (paste JSON between the backticks):

localStorage.setItem('vf-issue-tracker', `PASTE_HERE`); location.reload();
